## 📱 Mobile & Web Integration Examples

### For Web Apps (HTML/React/Vue)
```html
<iframe 
  src="https://your-app.streamlit.app" 
  width="100%" 
  height="600px"
  frameborder="0">
</iframe>
```

### For Mobile Apps (React Native/Flutter)
```javascript
// React Native WebView
<WebView 
  source={{ uri: 'https://your-app.streamlit.app' }}
  style={{ flex: 1 }}
/>
```
